# Icon

O componente Icon fornece uma maneira fácil de usar ícones da biblioteca Lucide React Native, com suporte para personalização de tamanho, cor e interações.

## Importação

```tsx
import { Icon } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `name` | `IconName` | - | Nome do ícone da biblioteca Lucide React Native |
| `size` | `number` | `24` | Tamanho do ícone em pixels |
| `color` | `string` | `theme.color.text` | Cor do ícone |
| `strokeWidth` | `number` | `1.5` | Espessura do traço do ícone |
| `style` | `ViewStyle` | - | Estilos personalizados para o contêiner do ícone |
| `onPress` | `() => void` | - | Função chamada quando o ícone é pressionado |
| `disabled` | `boolean` | `false` | Quando `true`, desabilita a interação com o ícone |
| `testID` | `string` | - | ID para testes automatizados |
| `accessibilityLabel` | `string` | - | Rótulo de acessibilidade para leitores de tela |
| `accessibilityHint` | `string` | - | Dica de acessibilidade para leitores de tela |
| `accessibilityRole` | `'button' \| 'image' \| 'none'` | `onPress ? 'button' : 'image'` | Papel de acessibilidade do componente |

## Exemplos

### Ícone Básico

```tsx
import React from 'react';
import { Icon } from '@s2mangas/native';

function ExemploIconeBasico() {
  return (
    <Icon name="Heart" />
  );
}
```

### Ícone Personalizado

```tsx
import React from 'react';
import { Icon } from '@s2mangas/native';

function ExemploIconePersonalizado() {
  return (
    <Icon 
      name="Star" 
      size={32} 
      color="#FFD700" 
      strokeWidth={2}
    />
  );
}
```

### Ícone com Ação

```tsx
import React from 'react';
import { View, Alert } from 'react-native';
import { Icon } from '@s2mangas/native';

function ExemploIconeComAcao() {
  const handlePress = () => {
    Alert.alert('Ícone pressionado!');
  };
  
  return (
    <Icon 
      name="Bell" 
      onPress={handlePress} 
      accessibilityLabel="Notificações"
      accessibilityHint="Pressione para ver suas notificações"
    />
  );
}
```

### Ícone Desabilitado

```tsx
import React from 'react';
import { Icon } from '@s2mangas/native';

function ExemploIconeDesabilitado() {
  return (
    <Icon 
      name="Trash" 
      onPress={() => console.log('Esta ação está desabilitada')} 
      disabled={true}
      color="#FF0000"
    />
  );
}
```

### Múltiplos Ícones

```tsx
import React from 'react';
import { View } from 'react-native';
import { Icon } from '@s2mangas/native';

function ExemploMultiplosIcones() {
  return (
    <View style={{ flexDirection: 'row', gap: 16 }}>
      <Icon name="Home" />
      <Icon name="Search" />
      <Icon name="User" />
      <Icon name="Settings" />
    </View>
  );
}
```

## Comportamento

O componente Icon possui as seguintes características de comportamento:

1. Renderiza automaticamente o ícone correspondente da biblioteca Lucide React Native
2. Quando `onPress` é fornecido, o ícone se torna clicável e é envolvido em um componente Pressable
3. Quando `disabled` é `true`, a opacidade é reduzida para 0.5 e a interação é desabilitada
4. Se o nome do ícone fornecido não existir na biblioteca, o componente não renderiza nada
5. O componente centraliza automaticamente o ícone dentro do seu contêiner

## Acessibilidade

O componente Icon implementa as seguintes características de acessibilidade:

- Define automaticamente `accessibilityRole` como "button" quando clicável ou "image" quando estático
- Se `accessibilityLabel` não for fornecido, usa o nome do ícone como rótulo padrão
- Suporta `accessibilityHint` para fornecer instruções adicionais para leitores de tela
- Quando desabilitado, a propriedade `disabled` é passada para o componente Pressable

## Personalização

O Icon pode ser personalizado através das propriedades de estilo, tamanho e cor:

```tsx
<Icon
  name="Heart"
  size={48}
  color="#FF0000"
  strokeWidth={3}
  style={{ 
    backgroundColor: '#F5F5F5', 
    padding: 8, 
    borderRadius: 24,
    elevation: 2
  }}
/>
```

## Lista de Ícones Disponíveis

O componente Icon utiliza a biblioteca Lucide React Native, que oferece uma ampla variedade de ícones. Você pode consultar a lista completa de ícones disponíveis na [documentação oficial do Lucide](https://lucide.dev/icons/).

Para usar um ícone, basta passar o nome do ícone (com a primeira letra maiúscula) para a propriedade `name`:

```tsx
<Icon name="Heart" />
<Icon name="ChevronRight" />
<Icon name="ShoppingCart" />
```
