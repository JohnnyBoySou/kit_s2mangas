# Skeleton

O componente Skeleton fornece uma representação visual de carregamento para conteúdo que ainda não está disponível, com efeito de brilho animado para melhorar a experiência do usuário durante estados de carregamento.

## Importação

```tsx
import { Skeleton } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `w` | `number \| string` | `0` | Largura do skeleton (pode ser número ou porcentagem como string) |
| `h` | `number \| string` | `0` | Altura do skeleton (pode ser número ou porcentagem como string) |
| `r` | `number` | - | Raio da borda do skeleton |
| `c` | `string` | `"#303030"` | Cor de fundo do skeleton |
| `style` | `ViewStyle` | - | Estilos personalizados para o skeleton |
| `testID` | `string` | - | ID para testes automatizados |

## Exemplos

### Skeleton Básico

```tsx
import React from 'react';
import { Skeleton, Column } from '@s2mangas/native';

function ExemploSkeletonBasico() {
  return (
    <Column>
      <Skeleton w={200} h={20} r={4} />
    </Column>
  );
}
```

### Skeleton para Card

```tsx
import React from 'react';
import { Skeleton, Column, Row } from '@s2mangas/native';

function ExemploSkeletonCard() {
  return (
    <Column style={{ width: 300, padding: 16, backgroundColor: '#1A1A1A', borderRadius: 8 }}>
      {/* Cabeçalho do card */}
      <Row>
        <Skeleton w={40} h={40} r={20} />
        <Column ml={12}>
          <Skeleton w={120} h={16} r={4} />
          <Skeleton w={80} h={12} r={4} mt={8} />
        </Column>
      </Row>
      
      {/* Corpo do card */}
      <Skeleton w="100%" h={120} r={4} mt={16} />
      
      {/* Rodapé do card */}
      <Row mt={16} justify="space-between">
        <Skeleton w={80} h={24} r={4} />
        <Skeleton w={60} h={24} r={4} />
      </Row>
    </Column>
  );
}
```

### Skeleton com Dimensões Responsivas

```tsx
import React from 'react';
import { Skeleton, Column } from '@s2mangas/native';

function ExemploSkeletonResponsivo() {
  return (
    <Column>
      <Skeleton w="90%" h={20} r={4} />
      <Skeleton w="75%" h={20} r={4} mt={8} />
      <Skeleton w="80%" h={20} r={4} mt={8} />
    </Column>
  );
}
```

### Skeleton com Cor Personalizada

```tsx
import React from 'react';
import { Skeleton, Column } from '@s2mangas/native';

function ExemploSkeletonColorido() {
  return (
    <Column>
      <Skeleton w={200} h={20} r={4} c="#424242" />
      <Skeleton w={150} h={20} r={4} mt={8} c="#424242" />
      <Skeleton w={180} h={20} r={4} mt={8} c="#424242" />
    </Column>
  );
}
```

## Comportamento

- O componente Skeleton exibe automaticamente um efeito de brilho animado que se move da esquerda para a direita.
- A animação é executada em loop contínuo até que o conteúdo real seja carregado e o Skeleton seja substituído.
- As dimensões podem ser especificadas como valores numéricos (pixels) ou como strings de porcentagem (ex: "90%").
- Quando uma dimensão é especificada como porcentagem, ela é calculada em relação à largura da janela.
- A animação é cancelada automaticamente quando o componente é desmontado para evitar vazamentos de memória.

## Acessibilidade

- O componente Skeleton é puramente visual e não possui interações diretas com o usuário.
- Recomenda-se usar o atributo `testID` para facilitar testes automatizados.
- Para melhorar a acessibilidade, considere usar componentes como `AccessibilityInfo` para informar leitores de tela sobre o estado de carregamento.

## Personalização

- A cor de fundo pode ser personalizada através da propriedade `c`.
- O componente aceita estilos adicionais através da propriedade `style`.
- O raio da borda pode ser ajustado com a propriedade `r` para corresponder ao design do seu aplicativo.
- A cor do efeito de brilho é definida internamente como "rgba(255, 255, 255, 0.15)" e não pode ser personalizada diretamente através de propriedades.
