# Input

O componente Input fornece uma interface para entrada de texto com suporte a diferentes estados, validação, máscaras e ícones.

## Importação

```tsx
import { Input } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `value` | `string` | `''` | Valor atual do input |
| `onChangeText` | `(text: string) => void` | - | Função chamada quando o texto é alterado |
| `label` | `string` | - | Texto do rótulo acima do input |
| `error` | `string` | - | Mensagem de erro a ser exibida abaixo do input |
| `helperText` | `string` | - | Texto de ajuda a ser exibido abaixo do input |
| `mask` | `MaskType` | - | Máscara para formatação do texto (importada de @s2mangas/core) |
| `keyboardType` | `KeyboardTypeOptions` | `'default'` | Tipo de teclado a ser exibido |
| `onSubmitEditing` | `() => void` | - | Função chamada quando o usuário pressiona o botão de envio no teclado |
| `secure` | `boolean` | `false` | Quando `true`, oculta o texto digitado (para senhas) |
| `disabled` | `boolean` | `false` | Quando `true`, desativa o input |
| `focused` | `boolean` | `false` | Quando `true`, o input recebe foco automaticamente |
| `required` | `boolean` | `false` | Quando `true`, exibe um asterisco vermelho ao lado do rótulo |
| `containerStyle` | `ViewStyle` | - | Estilos personalizados para o contêiner do input |
| `inputStyle` | `TextStyle` | - | Estilos personalizados para o campo de texto |
| `labelStyle` | `TextStyle` | - | Estilos personalizados para o rótulo |
| `errorStyle` | `TextStyle` | - | Estilos personalizados para a mensagem de erro |
| `helperStyle` | `TextStyle` | - | Estilos personalizados para o texto de ajuda |
| `testID` | `string` | - | ID para testes automatizados |
| `iconLeft` | `string` | - | Nome do ícone a ser exibido à esquerda do input |
| `iconRight` | `string` | - | Nome do ícone a ser exibido à direita do input |

## Métodos Imperativos

O componente Input pode ser controlado imperativamente através de uma ref:

```tsx
export type InputBigRef = {
  focus: () => void;
  blur: () => void;
  clear: () => void;
  getNode: () => TextInput | null;
};
```

## Exemplos

### Input Básico

```tsx
<Input
  label="Nome"
  value={nome}
  onChangeText={setNome}
  placeholder="Digite seu nome"
/>
```

### Com Validação de Erro

```tsx
<Input
  label="Email"
  value={email}
  onChangeText={setEmail}
  error={!isValidEmail(email) ? "Email inválido" : undefined}
  keyboardType="email-address"
  placeholder="exemplo@email.com"
/>
```

### Input de Senha

```tsx
<Input
  label="Senha"
  value={senha}
  onChangeText={setSenha}
  secure={true}
  placeholder="Digite sua senha"
/>
```

### Com Ícones

```tsx
<Input
  label="Pesquisar"
  value={pesquisa}
  onChangeText={setPesquisa}
  iconLeft="Search"
  placeholder="Buscar..."
/>

<Input
  label="Website"
  value={website}
  onChangeText={setWebsite}
  iconLeft="Globe"
  iconRight="ExternalLink"
  placeholder="https://exemplo.com"
/>
```

### Estados do Input

```tsx
<Input
  label="Input Desabilitado"
  value="Não editável"
  disabled={true}
/>

<Input
  label="Input com Foco"
  value={valorComFoco}
  onChangeText={setValorComFoco}
  focused={true}
/>

<Input
  label="Campo Obrigatório"
  value={campoObrigatorio}
  onChangeText={setCampoObrigatorio}
  required={true}
/>
```

### Com Texto de Ajuda

```tsx
<Input
  label="Username"
  value={username}
  onChangeText={setUsername}
  helperText="Seu nome de usuário deve ter entre 3 e 20 caracteres"
  placeholder="@username"
/>
```

## Acessibilidade

O componente Input implementa as seguintes características de acessibilidade:

- Utiliza o elemento nativo `TextInput` com propriedades de acessibilidade
- Suporta estados de desabilitado e foco
- Fornece feedback visual para estados de erro
- Inclui indicação clara para campos obrigatórios

## Personalização

O Input pode ser personalizado através das propriedades de estilo:

```tsx
<Input
  label="Input Personalizado"
  value={valorPersonalizado}
  onChangeText={setValorPersonalizado}
  containerStyle={{
    borderRadius: 12,
    borderColor: '#6200ee',
    borderWidth: 2
  }}
  labelStyle={{
    color: '#6200ee',
    fontSize: 14,
    fontWeight: 'bold'
  }}
  inputStyle={{
    fontSize: 16,
    color: '#333333'
  }}
/>
```

## Uso com Ref

```tsx
import { useRef } from 'react';
import { Input, InputBigRef } from '@s2mangas/native';

function MeuFormulario() {
  const inputRef = useRef<InputBigRef>(null);
  
  const focarInput = () => {
    inputRef.current?.focus();
  };
  
  const limparInput = () => {
    inputRef.current?.clear();
  };
  
  return (
    <>
      <Input
        ref={inputRef}
        label="Nome"
        value={nome}
        onChangeText={setNome}
      />
      <Button onPress={focarInput}>Focar</Button>
      <Button onPress={limparInput}>Limpar</Button>
    </>
  );
}
```
