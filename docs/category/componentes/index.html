Componentes

Todos os componentes do Kit

📄️ Button

O componente Button fornece uma interface interativa para ações do usuário, com suporte a diferentes variantes, tamanhos e estados.

📄️ Input

O componente Input fornece uma interface para entrada de texto com suporte a diferentes estados, validação, máscaras e ícones.

📄️ Switch

O componente Switch fornece uma interface para alternar entre dois estados (ligado/desligado), com animação suave e suporte a acessibilidade.

📄️ Avatar

O componente Avatar fornece uma interface para exibição de imagens de perfil, iniciais ou ícones em formato circular ou quadrado.

📄️ Card

O componente Card fornece um contêiner flexível para agrupar e exibir conteúdo, com suporte para personalização de bordas, sombras e espaçamento.

📄️ Checkbox

O componente Checkbox fornece uma interface para seleção binária, permitindo ao usuário marcar ou desmarcar uma opção com animações fluidas.

📄️ Badge

O componente Badge fornece uma forma visual de destacar informações, rótulos ou status, com suporte para diferentes variantes, tamanhos e interações.

📄️ Icon

O componente Icon fornece uma maneira fácil de usar ícones da biblioteca Lucide React Native, com suporte para personalização de tamanho, cor e interações.

📄️ Layout

Os componentes de Layout fornecem estruturas flexíveis para organizar elementos na interface, com suporte para diferentes direções, alinhamentos e espaçamentos.

📄️ Image

O componente Image fornece uma maneira fácil de exibir imagens em aplicativos React Native, com suporte para personalização, fallback, e otimizações de carregamento usando o Expo Image.

📄️ Skeleton

O componente Skeleton fornece uma representação visual de carregamento para conteúdo que ainda não está disponível, com efeito de brilho animado para melhorar a experiência do usuário durante estados de carregamento.

📄️ Radio

O componente Radio fornece uma representação visual de um botão de opção (radio button), com animações suaves para indicar o estado de seleção.

📄️ Select

O componente Select permite a seleção de uma opção a partir de uma lista suspensa, oferecendo uma interface intuitiva para escolhas em formulários.

📄️ Tabs

O componente Tabs permite a navegação entre diferentes seções de conteúdo através de abas interativas, oferecendo uma experiência de usuário organizada e intuitiva.

📄️ Text

Os componentes de texto fornecem diferentes estilos tipográficos para exibir conteúdo textual na aplicação, com suporte para várias variantes, tamanhos, cores e alinhamentos.

📄️ Loader

O componente Loader exibe um indicador de carregamento animado, útil para informar aos usuários que uma operação está em andamento.
