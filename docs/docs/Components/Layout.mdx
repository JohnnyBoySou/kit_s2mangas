# Layout

Os componentes de Layout fornecem estruturas flexíveis para organizar elementos na interface, com suporte para diferentes direções, alinhamentos e espaçamentos.

## Importação

```tsx
import { Column, Row, Main, ScrollHorizontal, ScrollVertical } from '@s2mangas/native';
```

## Componentes

### Column

Um contêiner que organiza elementos em uma coluna vertical.

#### Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `children` | `React.ReactNode` | - | Conteúdo a ser renderizado dentro da coluna |
| `style` | `ViewStyle` | - | Estilos personalizados para o contêiner |
| `justify` | `'flex-start' \| 'center' \| 'flex-end' \| 'space-between' \| 'space-around' \| 'space-evenly'` | `'flex-start'` | Alinhamento vertical dos elementos |
| `align` | `'flex-start' \| 'center' \| 'flex-end' \| 'stretch' \| 'baseline'` | `'stretch'` | Alinhamento horizontal dos elementos |
| `mh` | `number` | `0` | Margem horizontal |
| `mv` | `number` | `0` | Margem vertical |
| `mt` | `number` | `0` | Margem superior |
| `mb` | `number` | `0` | Margem inferior |
| `ml` | `number` | `0` | Margem esquerda |
| `mr` | `number` | `0` | Margem direita |
| `ph` | `number` | `0` | Padding horizontal |
| `pv` | `number` | `0` | Padding vertical |
| `pt` | `number` | `0` | Padding superior |
| `pb` | `number` | `0` | Padding inferior |
| `pl` | `number` | `0` | Padding esquerdo |
| `pr` | `number` | `0` | Padding direito |
| `gv` | `number` | `0` | Espaçamento vertical entre elementos (gap) |
| `gh` | `number` | `0` | Espaçamento horizontal entre elementos (gap) |
| `testID` | `string` | - | ID para testes automatizados |

### Main

Um contêiner principal que ocupa todo o espaço disponível na tela.

#### Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `children` | `React.ReactNode` | `<Text>Hello World</Text>` | Conteúdo a ser renderizado dentro do contêiner |
| `style` | `ViewStyle` | `{ flex: 1, backgroundColor: theme.color.background, paddingTop: 0 }` | Estilos personalizados para o contêiner |
| `testID` | `string` | - | ID para testes automatizados |

### ScrollHorizontal

Um contêiner com rolagem horizontal.

#### Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `children` | `React.ReactNode` | - | Conteúdo a ser renderizado dentro do scroll |
| `style` | `ViewStyle` | `{}` | Estilos personalizados para o contêiner |
| `contentContainerStyle` | `ViewStyle` | `{}` | Estilos para o contêiner de conteúdo do scroll |
| `pagingEnabled` | `boolean` | `false` | Quando `true`, o scroll se ajusta a páginas inteiras |
| `testID` | `string` | - | ID para testes automatizados |

### ScrollVertical

#### Propriedades

| `style` | `ViewStyle` | `{}` | Estilos personalizados para o contêiner |
| `contentContainerStyle` | `ViewStyle` | `{}` | Estilos para o contêiner de conteúdo do scroll |

## Exemplos

### Uso Básico de Column e Row

```tsx
import React from 'react';
import { Column, Row, Text } from '@s2mangas/native';

function ExemploLayout() {
  return (
    <Column>
      <Text>Coluna com itens empilhados verticalmente</Text>
      
      <Row mt={16}>
        <Text>Item 1</Text>
        <Text>Item 2</Text>
        <Text>Item 3</Text>
      </Row>
    </Column>
  );
}
```

### Alinhamento e Espaçamento

```tsx
import React from 'react';
import { Column, Row, Text } from '@s2mangas/native';

function ExemploAlinhamento() {
  return (
    <Column>
      <Row justify="space-between" align="center" ph={16} pv={8}>
        <Text>Esquerda</Text>
        <Text>Centro</Text>
        <Text>Direita</Text>
      </Row>
      
      <Column align="center" justify="center" mt={16} style={{ height: 200 }}>
        <Text>Centralizado</Text>
      </Column>
    </Column>
  );
}
```

### Uso de Main com ScrollVertical

```tsx
import React from 'react';
import { Main, ScrollVertical, Text } from '@s2mangas/native';

function ExemploMainScroll() {
  return (
    <Main>
      <ScrollVertical>
        <Text>Conteúdo com rolagem vertical</Text>
        {/* Mais conteúdo aqui */}
      </ScrollVertical>
    </Main>
  );
}
```

### ScrollHorizontal com Paginação

```tsx
import React from 'react';
import { ScrollHorizontal, Column, Text } from '@s2mangas/native';

function ExemploScrollPaginado() {
  return (
    <ScrollHorizontal pagingEnabled={true}>
      <Column style={{ width: 300, height: 200 }} justify="center" align="center">
        <Text>Página 1</Text>
      </Column>
      <Column style={{ width: 300, height: 200 }} justify="center" align="center">
        <Text>Página 2</Text>
      </Column>
      <Column style={{ width: 300, height: 200 }} justify="center" align="center">
        <Text>Página 3</Text>
      </Column>
    </ScrollHorizontal>
  );
}
```

## Comportamento

- Os componentes `Column` e `Row` utilizam flexbox para organizar elementos em diferentes direções.
- O componente `Main` é ideal para contêineres de nível superior, ocupando toda a tela.
- Os componentes `ScrollHorizontal` e `ScrollVertical` gerenciam automaticamente a rolagem e ocultam as barras de rolagem para uma interface mais limpa.
- As propriedades de margem e padding podem ser definidas de forma abreviada (`mh`, `mv`) ou específica (`mt`, `mb`, `ml`, `mr`).
- As propriedades de gap (`gv`, `gh`) permitem espaçamento consistente entre elementos filhos.

## Acessibilidade

- Todos os componentes aceitam a propriedade `testID` para testes automatizados.
- Os componentes de layout são neutros em termos de acessibilidade, mas fornecem a estrutura para organizar elementos acessíveis.

## Personalização

- Todos os componentes aceitam a propriedade `style` para personalização adicional.
- Os componentes de scroll também aceitam `contentContainerStyle` para estilizar o contêiner interno.
- O componente `Main` tem um estilo padrão que pode ser sobrescrito conforme necessário.
