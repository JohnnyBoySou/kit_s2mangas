S2Mangás Kit

S2Mangás Kit é uma ferramenta para ajudar a desenvolver interfaces mais rapidamente

Começar - 5min ⏱️

Componentes Prontos para React Native

O @s2mangas/native fornece componentes de UI reutilizáveis como Button, Switch, Input e Loader, prontos para usar em qualquer aplicação React Native.

Tema Consistente

O @s2mangas/core fornece cores, espaçamentos, tamanhos e outras variáveis de estilo, garantindo consistência visual em todos os componentes e telas do seu app.

Produtividade e Flexibilidade

Com o S2Mangás Kit, você economiza tempo e mantém a qualidade do projeto, usando apenas os pacotes que precisa, de forma modular e fácil de personalizar.
