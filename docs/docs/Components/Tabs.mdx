# Tabs

O componente Tabs permite a navegação entre diferentes seções de conteúdo através de abas interativas, oferecendo uma experiência de usuário organizada e intuitiva.

## Importação

```tsx
import { Tabs } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `value` | `string` | - | Valor da aba atualmente selecionada |
| `onChange` | `(value: string) => void` | - | Função chamada quando uma aba é selecionada |
| `values` | `string[]` | - | Lista de valores/rótulos para as abas |
| `testID` | `string` | - | ID para testes automatizados |
| `accessibilityRole` | `"button" \| "text"` | - | Função de acessibilidade do componente |
| `accessibilityLabel` | `string` | - | Rótulo de acessibilidade para leitores de tela |
| `accessibilityHint` | `string` | - | Dica de acessibilidade para leitores de tela |

## Exemplos

### Uso Básico

```tsx
import React, { useState } from 'react';
import { Tabs } from '@s2mangas/native';

const MyComponent = () => {
  const [selectedTab, setSelectedTab] = useState('Tab 1');
  
  return (
    <Tabs
      value={selectedTab}
      onChange={setSelectedTab}
      values={['Tab 1', 'Tab 2', 'Tab 3']}
    />
  );
};
```

### Com Conteúdo Condicional

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Tabs, Text } from '@s2mangas/native';

const TabContent = () => {
  const [selectedTab, setSelectedTab] = useState('Início');
  
  return (
    <View>
      <Tabs
        value={selectedTab}
        onChange={setSelectedTab}
        values={['Início', 'Perfil', 'Configurações']}
      />
      
      {selectedTab === 'Início' && (
        <Text>Conteúdo da aba Início</Text>
      )}
      
      {selectedTab === 'Perfil' && (
        <Text>Conteúdo da aba Perfil</Text>
      )}
      
      {selectedTab === 'Configurações' && (
        <Text>Conteúdo da aba Configurações</Text>
      )}
    </View>
  );
};
```

### Com Acessibilidade

```tsx
<Tabs
  value={selectedTab}
  onChange={setSelectedTab}
  values={['Início', 'Perfil', 'Configurações']}
  accessibilityLabel="Navegação principal"
  accessibilityHint="Toque para alternar entre as abas"
  accessibilityRole="button"
/>
```

## Comportamento

- As abas são exibidas horizontalmente em um contêiner com rolagem
- A aba selecionada é destacada visualmente com uma cor de fundo diferente
- Ao selecionar uma aba, ela é animada com um efeito de escala
- Quando uma aba é selecionada, o valor correspondente é passado para a função `onChange`

## Animações

O componente Tabs inclui as seguintes animações:

- **Escala**: As abas são animadas com um efeito de escala quando selecionadas ou desselecionadas
- **Transição suave**: A transição entre estados de seleção é suavizada com animações de mola (spring)

## Acessibilidade

O componente Tabs implementa as seguintes características de acessibilidade:

- Suporte para `accessibilityLabel` para descrever o propósito das abas para leitores de tela
- Utiliza `accessibilityRole` para indicar a função do componente
- Fornece `accessibilityHint` para explicar o resultado da interação
- Feedback visual claro para a aba selecionada

## Personalização

O componente Tabs possui um design predefinido com cores e estilos específicos. Para personalização mais avançada, considere criar um componente personalizado baseado no código-fonte do Tabs.
