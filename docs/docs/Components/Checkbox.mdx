# Checkbox

O componente Checkbox fornece uma interface para seleção binária, permitindo ao usuário marcar ou desmarcar uma opção com animações fluidas.

## Importação

```tsx
import { CheckBox } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `value` | `boolean` | - | Estado atual do checkbox (marcado/desmarcado) |
| `setValue` | `(value: boolean) => void` | - | Função para atualizar o estado do checkbox |
| `label` | `string` | - | Texto a ser exibido ao lado do checkbox |
| `testID` | `string` | - | ID para testes automatizados |
| `accessibilityLabel` | `string` | - | Rótulo de acessibilidade para leitores de tela |
| `accessibilityRole` | `'checkbox' \| 'switch'` | - | Papel de acessibilidade do componente |
| `accessibilityState` | `{ checked?: boolean; disabled?: boolean; }` | - | Estado de acessibilidade do componente |
| `disabled` | `boolean` | `false` | Quando `true`, desativa o checkbox |

## Exemplos

### Checkbox Básico

```tsx
import React, { useState } from 'react';
import { CheckBox } from '@s2mangas/native';

function ExemploCheckboxBasico() {
  const [marcado, setMarcado] = useState(false);
  
  return (
    <CheckBox 
      value={marcado}
      setValue={setMarcado}
      label="Aceitar termos e condições"
    />
  );
}
```

### Checkbox Desabilitado

```tsx
import React from 'react';
import { CheckBox } from '@s2mangas/native';

function ExemploCheckboxDesabilitado() {
  return (
    <>
      <CheckBox 
        value={true}
        setValue={() => {}}
        label="Opção selecionada (desabilitada)"
        disabled={true}
      />
      <CheckBox 
        value={false}
        setValue={() => {}}
        label="Opção não selecionada (desabilitada)"
        disabled={true}
      />
    </>
  );
}
```

### Checkbox com Acessibilidade

```tsx
import React, { useState } from 'react';
import { CheckBox } from '@s2mangas/native';

function ExemploCheckboxAcessivel() {
  const [marcado, setMarcado] = useState(false);
  
  return (
    <CheckBox 
      value={marcado}
      setValue={setMarcado}
      label="Receber notificações"
      accessibilityLabel="Ativar ou desativar notificações"
      accessibilityRole="checkbox"
    />
  );
}
```

## Comportamento

O componente Checkbox possui as seguintes características de comportamento:

1. Animações fluidas ao alternar entre os estados marcado/desmarcado
2. Transição suave de cores de fundo e texto
3. Animação de escala para o ícone de marcação
4. Feedback visual claro sobre o estado atual
5. Quando desabilitado, apresenta opacidade reduzida

## Acessibilidade

O componente Checkbox implementa as seguintes características de acessibilidade:

- Suporta `accessibilityLabel` para fornecer descrição para leitores de tela
- Permite definir `accessibilityRole` como "checkbox" ou "switch"
- Comunica o estado atual (marcado/desmarcado) através de `accessibilityState`
- Fornece feedback visual claro sobre o estado desabilitado
- Mantém contraste adequado entre texto e fundo

## Personalização

O Checkbox utiliza cores do tema e animações predefinidas, mas pode ser personalizado através do tema global da aplicação. As cores principais utilizadas são:

- Cor primária do tema para o estado marcado
- Cores de texto que se adaptam ao estado atual

Para personalizar o componente, você pode modificar as cores do tema em sua aplicação.
