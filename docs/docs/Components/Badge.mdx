# Badge

O componente Badge fornece uma forma visual de destacar informações, rótulos ou status, com suporte para diferentes variantes, tamanhos e interações.

## Importação

```tsx
import { Badge } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `children` | `React.ReactNode` | - | Conteúdo a ser renderizado dentro do badge |
| `label` | `string` | - | Texto a ser exibido no badge |
| `variant` | `'default' \| 'secondary' \| 'destructive' \| 'outline' \| 'ghost'` | `'default'` | Variante de estilo do badge |
| `size` | `'sm' \| 'default' \| 'md' \| 'lg'` | `'default'` | Tamanho do badge |
| `color` | `string` | - | Cor do texto do badge |
| `backgroundColor` | `string` | - | Cor de fundo do badge |
| `borderColor` | `string` | - | Cor da borda do badge |
| `style` | `ViewStyle` | - | Estilos personalizados para o badge |
| `textStyle` | `TextStyle` | - | Estilos personalizados para o texto do badge |
| `onPress` | `() => void` | - | Função chamada quando o badge é pressionado |
| `onRemove` | `() => void` | - | Função para adicionar um botão de remoção ao badge |
| `testID` | `string` | - | ID para testes automatizados |
| `accessibilityLabel` | `string` | - | Rótulo de acessibilidade para leitores de tela |
| `accessibilityRole` | `'button' \| 'text'` | - | Papel de acessibilidade do componente |
| `icon` | `React.ReactNode` | - | Ícone a ser exibido no badge |
| `selected` | `boolean` | `false` | Quando `true`, aplica estilo de seleção ao badge |


### Badge Removível

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Badge } from '@s2mangas/native';

function ExemploBadgeRemovivel() {
  const [badges, setBadges] = useState(['React', 'React Native', 'TypeScript']);
  
  const removerBadge = (index) => {
    setBadges(badges.filter((_, i) => i !== index));
  };
  
  return (
    <View style={{ flexDirection: 'row', flexWrap: 'wrap', gap: 8 }}>
      {badges.map((badge, index) => (
        <Badge 
          key={index}
          label={badge}
          onRemove={() => removerBadge(index)}
        />
      ))}
    </View>
  );
}
```

### Badge Interativo

```tsx
import React, { useState } from 'react';
import { View } from 'react-native';
import { Badge } from '@s2mangas/native';

function ExemploBadgeInterativo() {
  const [selecionados, setSelecionados] = useState([]);
  
  const toggleSelecao = (tag) => {
    if (selecionados.includes(tag)) {
      setSelecionados(selecionados.filter(item => item !== tag));
    } else {
      setSelecionados([...selecionados, tag]);
    }
  };
  
  const tags = ['React', 'Angular', 'Vue', 'Svelte', 'Next.js'];
  
  return (
    <View style={{ flexDirection: 'row', flexWrap: 'wrap', gap: 8 }}>
      {tags.map((tag) => (
        <Badge 
          key={tag}
          label={tag}
          onPress={() => toggleSelecao(tag)}
          selected={selecionados.includes(tag)}
          variant="outline"
        />
      ))}
    </View>
  );
}
```

## Comportamento

O componente Badge possui as seguintes características de comportamento:

1. Adapta-se automaticamente ao conteúdo, mantendo um tamanho mínimo baseado na propriedade `size`
2. Quando `onRemove` é fornecido, exibe um botão de remoção à direita do conteúdo
3. Quando `onPress` é fornecido, o badge inteiro se torna clicável
4. Quando `selected` é `true`, o texto é exibido em negrito e pode haver alterações visuais dependendo da variante
5. Quando um `icon` é fornecido, ele é exibido à esquerda do texto

## Acessibilidade

O componente Badge implementa as seguintes características de acessibilidade:

- Suporta `accessibilityLabel` para fornecer descrição para leitores de tela
- Define automaticamente `accessibilityRole` como "button" quando clicável ou "text" quando estático
- Mantém contraste adequado entre texto e fundo para todas as variantes
- O botão de remoção possui seu próprio rótulo de acessibilidade

## Personalização

O Badge pode ser personalizado através das propriedades de estilo e cores:

```tsx
<Badge
  label="Personalizado"
  backgroundColor="#6200ee"
  borderColor="#3700b3"
  color="#ffffff"
  textStyle={{ fontWeight: 'bold' }}
  style={{ shadowOpacity: 0.3, shadowRadius: 2, elevation: 2 }}
/>
```

As variantes disponíveis oferecem estilos predefinidos:

- `default`: Fundo colorido com a cor primária
- `secondary`: Fundo colorido com a cor secundária
- `destructive`: Fundo colorido com a cor destrutiva (geralmente vermelho)
- `outline`: Apenas borda, sem preenchimento
- `ghost`: Fundo semi-transparente da cor primária
