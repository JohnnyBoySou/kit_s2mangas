# Radio

O componente Radio fornece uma representação visual de um botão de opção (radio button), com animações suaves para indicar o estado de seleção.

## Importação

```tsx
import { Radio } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `selected` | `boolean` | `false` | Estado de seleção do componente Radio |

## Exemplos

### Radio Básico

```tsx
import React, { useState } from 'react';
import { Radio, Row, Text, Column, Pressable } from '@s2mangas/native';

function ExemploRadioBasico() {
  const [selected, setSelected] = useState(false);
  
  return (
    <Pressable onPress={() => setSelected(!selected)}>
      <Row align="center">
        <Radio selected={selected} />
        <Text ml={8}>Opção</Text>
      </Row>
    </Pressable>
  );
}
```

### Grupo de Radio Buttons

```tsx
import React, { useState } from 'react';
import { Radio, Row, Text, Column, Pressable } from '@s2mangas/native';

function ExemploGrupoRadio() {
  const [selectedOption, setSelectedOption] = useState('opcao1');
  const options = [
    { id: 'opcao1', label: 'Opção 1' },
    { id: 'opcao2', label: 'Opção 2' },
    { id: 'opcao3', label: 'Opção 3' },
  ];
  
  return (
    <Column>
      {options.map((option) => (
        <Pressable 
          key={option.id}
          onPress={() => setSelectedOption(option.id)}
          style={{ marginVertical: 8 }}
        >
          <Row align="center">
            <Radio selected={selectedOption === option.id} />
            <Text ml={8}>{option.label}</Text>
          </Row>
        </Pressable>
      ))}
    </Column>
  );
}
```

### Radio com Estilo Personalizado

```tsx
import React, { useState } from 'react';
import { Radio, Row, Text, Column, Pressable } from '@s2mangas/native';

function ExemploRadioPersonalizado() {
  const [selected, setSelected] = useState(false);
  
  return (
    <Pressable 
      onPress={() => setSelected(!selected)}
      style={{
        padding: 12,
        backgroundColor: selected ? 'rgba(255, 255, 255, 0.1)' : 'transparent',
        borderRadius: 8,
      }}
    >
      <Row align="center">
        <Radio selected={selected} />
        <Column ml={12}>
          <Text>Opção com descrição</Text>
          <Text style={{ fontSize: 12, opacity: 0.7 }} mt={4}>
            Descrição adicional para esta opção
          </Text>
        </Column>
      </Row>
    </Pressable>
  );
}
```

## Comportamento

- O componente Radio exibe um círculo externo com borda e um círculo interno que indica o estado de seleção.
- Quando o estado `selected` muda para `true`, o componente anima suavemente o círculo interno, aumentando seu tamanho e alterando sua cor.
- Quando o estado `selected` muda para `false`, o componente anima suavemente o círculo interno, diminuindo seu tamanho.
- A animação de seleção utiliza spring physics para um efeito mais natural, enquanto a animação de desseleção utiliza timing para uma transição mais suave.
- As cores são interpoladas durante a animação, transitando entre os estados selecionado e não selecionado.

## Acessibilidade

- O componente Radio por si só não inclui propriedades de acessibilidade, portanto é recomendável envolvê-lo em um componente `Pressable` ou `TouchableOpacity` com as propriedades de acessibilidade adequadas.
- Ao implementar um grupo de Radio buttons, certifique-se de adicionar rótulos de acessibilidade apropriados e garantir que os componentes possam ser navegados corretamente com tecnologias assistivas.

## Personalização

- O componente Radio tem um design fixo com cores predefinidas, mas pode ser personalizado visualmente ao ser combinado com outros componentes.
- Para criar uma experiência de usuário consistente, recomenda-se envolver o Radio em um componente personalizado que inclua o rótulo e o comportamento de pressionar.
- As cores do Radio são definidas internamente e não podem ser personalizadas diretamente através de propriedades. Para personalização avançada, considere criar uma versão modificada do componente.
