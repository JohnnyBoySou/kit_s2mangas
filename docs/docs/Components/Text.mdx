# Text

Os componentes de texto fornecem diferentes estilos tipográficos para exibir conteúdo textual na aplicação, com suporte para várias variantes, tamanhos, cores e alinhamentos.

## Importação

```tsx
import { HeadTitle, Title, Label, SubLabel, Description, Heading, U } from '@s2mangas/native';
```

## Variantes

O pacote oferece várias variantes de componentes de texto, cada uma com um propósito específico:

### HeadTitle

Componente para títulos principais, com ênfase visual máxima.

```tsx
<HeadTitle>Título Principal</HeadTitle>
```

### Title

Componente para títulos secundários e cabeçalhos de seção.

```tsx
<Title>Título da Seção</Title>
```

### Label

Componente para rótulos e textos de destaque médio.

```tsx
<Label>Rótulo do Campo</Label>
```

### SubLabel

Componente para rótulos secundários e textos de menor destaque.

```tsx
<SubLabel>Informação adicional</SubLabel>
```

### Description

Componente para textos descritivos e parágrafos.

```tsx
<Description>Este é um texto descritivo que explica alguma funcionalidade ou fornece informações adicionais ao usuário.</Description>
```

### Heading

Componente genérico para cabeçalhos com níveis configuráveis (h1-h6).

```tsx
<Heading level={2}>Título de Nível 2</Heading>
```

### U

Componente para texto sublinhado.

```tsx
<Label>Texto normal <U>texto sublinhado</U> texto normal</Label>
```

## Propriedades Comuns

Todas as variantes de texto compartilham as seguintes propriedades:

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `children` | `React.ReactNode` | - | Conteúdo de texto a ser exibido |
| `size` | `number` | Varia por componente | Tamanho da fonte |
| `align` | `'left' \| 'center' \| 'right'` | `'left'` | Alinhamento horizontal do texto |
| `color` | `string` | Varia por componente | Cor do texto |
| `mh` | `number` | `0` | Margem horizontal |
| `mv` | `number` | `0` | Margem vertical |
| `mb` | `number` | `0` | Margem inferior |
| `mt` | `number` | `0` | Margem superior |
| `mr` | `number` | `0` | Margem direita |
| `ml` | `number` | `0` | Margem esquerda |
| `fontFamily` | `string` | Varia por componente | Família da fonte |
| `spacing` | `number` | `0` | Espaçamento entre letras |
| `style` | `TextStyle` | - | Estilos personalizados adicionais |
| `level` | `1 \| 2 \| 3 \| 4 \| 5 \| 6` | - | Nível de cabeçalho (apenas para o componente Heading) |

## Exemplos

### Uso Básico

```tsx
import { HeadTitle, Title, Label, Description } from '@s2mangas/native';

const TextExample = () => (
  <>
    <HeadTitle>Título Principal</HeadTitle>
    <Title>Seção Importante</Title>
    <Label>Este é um rótulo</Label>
    <Description>Aqui está uma descrição detalhada sobre algum assunto relevante para o usuário.</Description>
  </>
);
```

### Com Estilos Personalizados

```tsx
<Title 
  size={24} 
  color="#007bff" 
  align="center"
  mb={10}
>
  Título Personalizado
</Title>
```

### Usando Níveis de Cabeçalho

```tsx
import { Heading } from '@s2mangas/native';

const HeadingExample = () => (
  <>
    <Heading level={1}>Título H1</Heading>
    <Heading level={2}>Título H2</Heading>
    <Heading level={3}>Título H3</Heading>
    <Heading level={4}>Título H4</Heading>
    <Heading level={5}>Título H5</Heading>
    <Heading level={6}>Título H6</Heading>
  </>
);
```

### Combinando Estilos

```tsx
<Label>
  Texto normal com <U>parte sublinhada</U> e continuação normal.
</Label>
```

## Comportamento

- Os componentes de texto se ajustam automaticamente ao espaço disponível
- O texto é quebrado automaticamente (wrap) quando excede a largura disponível
- A altura da linha é calculada proporcionalmente ao tamanho da fonte

## Acessibilidade

Os componentes de texto implementam as seguintes características de acessibilidade:

- Utilizam `accessibilityRole="header"` para componentes de título (HeadTitle, Title, Heading)
- Utilizam `accessibilityRole="text"` para componentes de texto regular (Label, SubLabel, Description)
- Extraem automaticamente o texto para `accessibilityLabel` quando possível
- Mantêm contraste adequado entre texto e fundo com as cores padrão

## Personalização

Os componentes de texto podem ser personalizados através das propriedades de estilo ou da propriedade `style` para ajustes mais específicos. É possível modificar tamanhos, cores, famílias de fonte e outros aspectos visuais para adequar os componentes ao design da aplicação.
