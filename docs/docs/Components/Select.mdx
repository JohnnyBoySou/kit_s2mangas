# Select

O componente Select permite a seleção de uma opção a partir de uma lista suspensa, oferecendo uma interface intuitiva para escolhas em formulários.

## Importação

```tsx
import { Select } from '@s2mangas/native';
```

## Propriedades

| Propriedade | Tipo | Padrão | Descrição |
| ----------- | ---- | ------ | --------- |
| `options` | `Array<{label: string, value: any}>` | - | Lista de opções disponíveis para seleção |
| `value` | `any` | - | Valor atualmente selecionado |
| `onChange` | `(value: any) => void` | - | Função chamada quando uma opção é selecionada |
| `placeholder` | `string` | `"Selecione uma opção"` | Texto exibido quando nenhuma opção está selecionada |
| `style` | `ViewStyle` | - | Estilos personalizados para o contêiner do select |
| `textStyle` | `TextStyle` | - | Estilos personalizados para o texto do select |
| `disabled` | `boolean` | `false` | Define se o select está desabilitado |
| `testID` | `string` | - | ID para testes automatizados |
| `accessibilityLabel` | `string` | - | Rótulo de acessibilidade para leitores de tela |
| `accessibilityRole` | `"button"` | - | Função de acessibilidade do componente |

## Exemplos

### Uso Básico

```tsx
import React, { useState } from 'react';
import { Select } from '@s2mangas/native';

const MyComponent = () => {
  const [selectedValue, setSelectedValue] = useState(null);
  
  const options = [
    { label: 'Opção 1', value: 1 },
    { label: 'Opção 2', value: 2 },
    { label: 'Opção 3', value: 3 },
  ];

  return (
    <Select
      options={options}
      value={selectedValue}
      onChange={setSelectedValue}
      placeholder="Selecione uma opção"
    />
  );
};
```

### Com Estilo Personalizado

```tsx
<Select
  options={options}
  value={selectedValue}
  onChange={setSelectedValue}
  style={{ borderColor: '#007bff', backgroundColor: '#f0f8ff' }}
  textStyle={{ color: '#007bff', fontWeight: 'bold' }}
/>
```

### Desabilitado

```tsx
<Select
  options={options}
  value={selectedValue}
  onChange={setSelectedValue}
  disabled={true}
/>
```

## Comportamento

- Ao clicar no componente, uma lista suspensa é exibida com todas as opções disponíveis
- A opção atualmente selecionada é destacada na lista
- Ao selecionar uma opção, a lista é fechada automaticamente
- Quando desabilitado, o componente aparece com opacidade reduzida e não responde a interações

## Acessibilidade

O componente Select implementa as seguintes características de acessibilidade:

- Suporte para `accessibilityLabel` para descrever o propósito do select para leitores de tela
- Utiliza `accessibilityRole="button"` para indicar que o elemento é interativo
- Fornece feedback visual para a opção selecionada

## Personalização

O componente Select pode ser personalizado através das propriedades `style` e `textStyle`, permitindo ajustes na aparência do contêiner e do texto. É possível modificar cores, bordas, preenchimento e outras propriedades de estilo para adequar o componente ao design da aplicação.
